# CRM/Archive/

Закрытые сделки — и выигранные (won), и проигранные (lost).

## Правила

- Папки переносятся целиком из `CRM/Active/` после закрытия
- В `deal.md` закрытой сделки должен быть заполнен блок `## Итог`:
  - Won: финальная сумма, срок реализации, что работало
  - Lost: причина (бюджет, время, выбрали конкурента), какие уроки
- **Не удаляй** папки — они нужны для ретро и похожих будущих сделок
- При запуске `/prospect-research` агенты ищут похожие кейсы именно в Archive
