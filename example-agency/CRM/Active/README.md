# CRM/Active/

Активные сделки. **1 папка = 1 сделка.**

## Именование папки

```
YYYY-MM-DD_название-клиента/
```

- `YYYY-MM-DD` — дата первого контакта
- `название-клиента` — slug в kebab-case (без пробелов, только латиница + цифры + дефисы)

Примеры: `2026-03-15_acme-corp/`, `2026-04-01_helloworld-startup/`

## Структура папки сделки

```
YYYY-MM-DD_клиент/
├── deal.md                  # ГЛАВНЫЙ файл — всегда читай первым
├── changelog.md             # Хронология изменений (append-only)
├── business-analysis.md     # Внутренний анализ (из /prospect-research)
├── briefing-for-partner.md  # Внешний документ (из /prospect-brief)
├── proposal.docx            # КП (из /create-proposal)
└── attachments/             # PDF, скриншоты, технические спеки, транскрипты встреч
```

## Шаблон deal.md

```markdown
# <Клиент> — <Тип сделки>

**Сумма:** X ₽ (или диапазон)
**Вероятность:** XX%
**Старт:** YYYY-MM-DD
**Дедлайн:** YYYY-MM-DD

## Контакты

| Имя | Роль | Контакт |
|-----|------|---------|
| ... | ЛПР | ... |

## Что делаем

Краткое описание решения (1-3 абзаца).

## Этапы

- [ ] Пилот

- [ ] Поддержка

## История встреч

- YYYY-MM-DD — тип встречи — короткое резюме (см. attachments/)
```
