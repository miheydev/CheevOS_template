# Docs/Strategy/

Стратегические документы агентства — не привязанные к конкретной сделке.

Что сюда кладём:
- **Лидген гипотезы** — что пробуем, что работает, что провалилось (трекер)
- **Позиционирование** — кто мы, что продаём, чем отличаемся
- **Конкуренты** — разбор ключевых игроков
- **Продуктовые линейки** — описание своих продуктов/услуг
- **Квартальные / годовые планы**
- **Ретро по сделкам** — агрегированные уроки из `CRM/Archive/`

Не сюда:
- Конкретные сделки → `CRM/Active/`
- Кейсы клиентов для публикации → `Docs/Content/`
- Ресёрчи по индустриям → `Docs/Research/`
