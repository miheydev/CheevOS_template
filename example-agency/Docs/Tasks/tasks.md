# Задачи агентства

> Командные задачи. Личные задачи — в `example-knowledge-base/Goals/` или твоём персональном трекере.

## Формат

Простой markdown чек-лист с приоритетом и ответственным:

```
- [ ] (P1) @<ответственный> Коротко что сделать — контекст / ссылка на сделку
```

Приоритеты:
- **P0** — пожар, сегодня
- **P1** — эта неделя
- **P2** — этот месяц
- **P3** — когда руки дойдут

## Активные задачи

- [ ] (P?) @<owner> Пример задачи

## Завершённые (на этой неделе)

- [x] ~~Пример завершённой задачи~~
