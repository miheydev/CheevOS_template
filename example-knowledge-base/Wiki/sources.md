# Sources Registry

> Реестр всех источников, которые были обработаны в wiki. Помогает AI не обрабатывать один и тот же источник дважды.
>
> Добавляй запись при каждом ingest.

## Формат записи

```markdown
### <YYYY-MM-DD> · <Название источника>
- **Type:** article / video / podcast / book / twitter / transcript
- **URL:** <url>
- **Raw file:** [raw/<filename>](./raw/<filename>)  <!-- если сохранён локально -->
- **Wiki pages:** [page-name.md](./page-name.md), [another.md](./another.md)
- **Tags:** #tag1 #tag2
```
