# Wiki/raw/

**Неизменяемая** папка с сырыми файлами. AI **никогда** не редактирует файлы здесь — только добавляет новые при ingest.

## Что сюда попадает

- Скачанные статьи (PDF, HTML, markdown)
- Транскрипты видео / подкастов
- Скриншоты и выжимки из Twitter / X
- Экспорты из других сервисов

## Именование

```
YYYY-MM-DD_<slug-источника>.<ext>
```

Пример: `2026-03-15_karpathy-llm-wiki-gist.md`, `2026-04-02_lex-fridman-altman-transcript.txt`
