# Brand/Content/tov-profiles/

Tone of Voice профили автора. Загружаются скиллом `/tov-write` для генерации текстов в заданном стиле.

## Как создать профиль

1. Собери корпус своих текстов (~20-30 постов, писем, сообщений) в один файл
2. Запусти Anthropic-скилл `tone-of-voice-analyzer` (если установлен) или `/content-ideas` с отдельным промптом
3. Сохрани результат сюда как `<канал>-tov.md`

## Структура профиля

Каждый профиль должен содержать минимум 3 слоя:

```markdown
# ToV: <канал / контекст>

## Style Card (короткая версия, для prompt)

<5-10 строк — ключевые маркеры голоса>

## One-Pager (средняя версия)

### Лексика
- Характерные слова
- Слова-маркеры ("короче", "прям", "ну и...")
- Слова-запреты

### Синтаксис
- Средняя длина предложения
- Типичная структура абзаца

### Приёмы
- Хуки
- Концовки
- Как начинает, как заканчивает

## Full Guide (длинная версия)

<полный разбор с примерами>
```

## Примеры профилей для разных каналов

- `personal-tg.md` — личный Telegram (разговорный, с самоиронией)
- `blog-expert.md` — экспертный блог (структурированный, с источниками)
- `linkedin.md` — LinkedIn (формальнее, но не корпоратив)
- `email-client.md` — деловые письма клиентам

## Правило

**Без профиля не генерируй.** Если пользователь просит "напиши пост в моём стиле", а профиля нет — сначала предложи создать профиль, не генерируй в дефолтном "экспертном" голосе.
