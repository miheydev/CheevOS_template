# Ideas/

Сырой слой. Быстрые заметки, гипотезы, полумысли — всё что ещё не дозрело до проекта.

## Правила

- **Одна идея = один файл.** Не копи идеи в один общий `ideas.md` — они тонут.
- **Именование:** `YYYY-MM-DD_<краткий-slug>.md`
- **Не полируй.** Идея в `Ideas/` может быть в виде буллетов, неполных предложений, вопросов самому себе. Это нормально.
- **Дозрела → в проект.** Если идея превратилась в план с метриками и дедлайном — **перемести** (не копируй) файл в `../Projects/Active/<slug>/` и дополни структуру.

## Шаблон (минимальный)

```markdown
# <Название>

> YYYY-MM-DD

## Что за идея

<1-3 абзаца>

## Почему это может сработать

-
-

## Что мешает

-
-

## Следующий микрошаг (опционально)

<что можно сделать за 30 минут, чтобы проверить идею>
```
