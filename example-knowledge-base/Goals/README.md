# Goals/

Личные цели — от годовых до месячных.

## Структура файлов

```
Goals/
├── YYYY-year-plan.md            # Годовой план (North Stars, ключевые результаты)
├── YYYY-QN-plan.md              # Квартальный
├── YYYY-MM-goals.md             # Месячные цели
└── retrospectives/
    ├── YYYY-MM-retro.md         # Ретро месяца
    └── YYYY-year-retro.md       # Годовое ретро
```

## Шаблон годового плана (YYYY-year-plan.md)

```markdown
# Год YYYY

## Слово года

<одно слово-фокус>

## North Stars (3 штуки, не больше)

1. **<Сфера>:** <измеримый результат>
2. ...

## Ключевые риски

- <риск> → <что делаем чтобы избежать>

## Квартальная разбивка

- Q1: <тема>
- Q2: <тема>
- Q3: <тема>
- Q4: <тема>
```

## Шаблон месячных целей (YYYY-MM-goals.md)

```markdown
# YYYY-MM

## Цели месяца (3-5, не больше)

1. [ ] <конкретная цель с метрикой>
2. ...

## Прогресс

- [x] <завершённое> (YYYY-MM-DD)
- [ ] <в процессе> — <блокер, если есть>
```

## Правила

- **Не больше 3-5 целей в месяце.** Всё остальное — не цели, а задачи. Они в `Projects/Active/` или в `example-agency/Docs/Tasks/`.
- **Цель без метрики = надежда.** Сразу прописывай как будешь измерять успех.
- **Ретро в конце месяца обязательно.** Без ретро цели не работают.
