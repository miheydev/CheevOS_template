# INBOX

Временная папка для входящих файлов (скриншоты, pdf, транскрипты, заметки), которые нужно распределить по проектам.

## Как пользоваться

1. Кидай файлы сюда, если пока не знаешь, куда их положить
2. Скажи в Claude Code: **"разбери INBOX"**
3. AI прочитает каждый файл и переместит его в нужную папку проекта

Подробный воркфлоу → [.claude/skills/inbox-root-processing.md](../.claude/skills/inbox-root-processing.md)

## Правила

- **Не архив.** После обработки файл должен уехать в проект, а не лежать здесь бесконечно
- **`mv`, не `cp`.** Перемещай, а не копируй — иначе INBOX быстро засорится
- **Не трогается автоматически.** Claude Code НЕ проверяет INBOX при старте сессии — только по явному запросу
