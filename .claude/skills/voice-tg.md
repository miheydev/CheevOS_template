# Skill: /voice-tg

> Голос автора для Telegram — превращает поток мыслей в готовое сообщение

## Триггеры

- `/voice-tg`
- "голос автора для телеграм"
- "голос автора для тг"
- "оформи для телеги"
- "подготовь для телеграма"

## Инструкция

Пользователь отправит войс (транскрипт) или поток мыслей в тексте. Твоя задача — **структурировать**, сохранив авторский голос.

### Алгоритм

1. **Прочитай** весь текст целиком
2. **Определи** смысловые блоки и их порядок
3. **Структурируй** в связный текст
4. **Оформи** в Telegram Markdown

---

## Жёсткие правила (без исключений)

| Правило | Что значит |
|---------|------------|
| **Только слова автора** | Не добавляй синонимы, метафоры, "улучшения" |
| **Неясно = дословно** | Если фрагмент непонятен — сохрани как есть |
| **Структурируй, не генерируй** | Твоя задача — упорядочить, а не написать заново |
| **Объём ≈ оригиналу** | Не сжимай до потери смысла, не раздувай |
| **Ничего от себя** | Никаких вводных, пояснений, комментариев |

### Стиль

- Естественно, по-человечески, без официоза
- Чётко и с уважением к собеседнику
- Если нужно согласовать пункты — оформи списком

---

## Формат вывода — Telegram Markdown

Telegram Desktop поддерживает Markdown. Используй:

```
**жирный**
__подчёркнутый__
*курсив* или _курсив_
~зачёркнутый~
`моноширинный`
||спойлер||
```

**Правила форматирования:**
- Заголовки и ключевые фразы — **жирным**
- Важные термины — *курсивом*
- Для списков: • или —
- Разбивай на абзацы пустой строкой
- Не злоупотребляй — выделяй только важное
