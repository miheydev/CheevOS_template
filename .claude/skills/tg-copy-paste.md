# Skill: /tg-copy-paste

> Подготовка текста для копирования в Telegram Desktop с сохранением форматирования

## Триггеры

- `/tg-copy-paste`
- "сделай для копирования в телегу"
- "подготовь для вставки в TG"
- "нужен текст с форматированием для телеграма"

---

## Как работает

Telegram Desktop поддерживает **жирный** (`**текст**`) при вставке из буфера. Гиперссылки не поддерживаются через markdown, но `@username` и `https://` ссылки автолинкуются самим Telegram.

## Алгоритм

1. Получи готовый текст (из другого скилла или напрямую от пользователя)
2. Выведи текст в **code block** (тройные бэктики) с Telegram-совместимым форматированием
3. Пользователь нажимает кнопку копирования на code block → вставляет в TG Desktop

---

## Правила форматирования

| Что нужно | Как писать | Пример |
|-----------|-----------|--------|
| Жирный | `**текст**` | **Заголовок** |
| Курсив | `__текст__` | __акцент__ |
| Моноширинный | `` `текст` `` | `код` |
| Ссылка | просто URL | https://example.com |
| Юзернейм | `@username` | @hegai_circle_bot |
| Список | тире текстом | — пункт 1 |
| Перенос строки | пустая строка | |

### Чего НЕ использовать

- `[текст](url)` — markdown-ссылки НЕ работают в TG при вставке
- `# заголовки` — не поддерживаются
- HTML-теги — не поддерживаются при вставке из буфера

---

## Пример

**Вход:** markdown-файл с постом

**Выход:** code block в чате Claude Code:

````
**Заголовок поста**

Первый абзац текста.

Второй абзац с **жирным** и @username.

— пункт 1
— пункт 2

Ссылка: https://example.com
````

**Инструкция пользователю:** Нажми кнопку копирования → вставь в Telegram Desktop.
