## Триггеры

- `/video-summary`
- "сделай конспект видео"
- "проанализируй воркшоп"
- "конспект лекции"
- "разбери доклад"

## Инструкция

### Входные данные

Пользователь даёт:
- YouTube URL — извлечь транскрипт через `/youtube-transcript`
- Или готовый транскрипт (файл / текст)
- Опционально: дополнительный контекст (ссылка на GitHub, сайт проекта, описание)

### Алгоритм

1. **Извлечь транскрипт** (если YouTube — через `youtube-transcript-api`)
2. **Получить метаданные** видео (заголовок, канал, дата, описание)
3. **Собрать дополнительный контекст** — если упоминаются сайты/репо, обогатить конспект данными оттуда
4. **Составить конспект** по структуре ниже
5. **Определить папку** для сохранения (спросить пользователя, если неочевидно)
6. **Сохранить** как markdown-файл

## Жёсткие правила

| Правило | Что значит |
|---------|------------|
| **Факты из транскрипта** | Не додумывай то, чего спикер не говорил |
| **Ссылки — только реальные** | Если URL не звучал явно — пометь как "показан на экране (не озвучен)" |
| **Структура > хронология** | Группируй по смыслу, а не "сначала он сказал, потом он сказал" |
| **Авто-субтитры ≠ точность** | Предупреждай о возможных ошибках распознавания |
| **Практические выводы** | Всегда добавляй чеклист "что делать дальше" |
