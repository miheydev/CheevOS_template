# Skill: /gen-image

> Генерация картинок через OpenAI `gpt-image-1.5`. Ключ тянется из `<your-knowledge-base>/.env` автоматически.

## Триггеры

- `/gen-image <prompt>`
- «сгенерируй картинку / нарисуй / сделай иллюстрацию / нужна картинка / создай изображение / сделай постер»

## Алгоритм

1. **Собери промпт.** Если пользователь дал короткое описание — расширь в английский prompt (OpenAI лучше работает с en): сцена → субъект → стиль → свет → композиция. Русский тоже работает, но en даёт более стабильный результат.
2. **Определи параметры:**
   - `--size`: `1024x1024` (квадрат, по умолчанию), `1536x1024` (горизонт), `1024x1536` (портрет/сторис), `auto`.
   - `--quality`: `high` (по умолчанию, медленнее), `medium`, `low` (быстро/дёшево), `auto`.
   - `--out`: путь. **По умолчанию `.claude/generated/`** (внутри workspace — ссылки кликабельны в VS Code). Если сказано «в сделку» / «в Downloads» — клади туда.
   - `-n`: количество вариантов (по умолчанию 1). Для подбора — 2–4.
3. **Запусти:**
   ```bash
   /usr/bin/python3 /Users/igor/Documents/CheevOS/.claude/scripts/gen_image.py "<prompt>" \
     --size 1024x1024 --quality high
   ```
   Скрипт печатает путь(-и) к сохранённым PNG последними строками stdout.
4. **Покажи превью** — вызови `Read` с абсолютным путём к PNG, чтобы картинка отрендерилась прямо в чате.
5. **Дай кликабельную ссылку** в формате `[имя.png](file:///абсолютный/путь.png)` — `~/` и tilde VS Code **не резолвит**, нужен `file://`-URL или путь относительно workspace (`./INBOX/foo.png`). Пробелы в пути — URL-encode (`%20`).
6. **Открой файл локально** через `open "<путь>"` (macOS) — откроется в Preview, удобно сразу посмотреть в полном размере.

## Параметры API

| Параметр | Значения |
|----------|----------|
| model | `gpt-image-1.5` (default), `gpt-image-1`, `gpt-image-1-mini` |
| size | `1024x1024`, `1536x1024`, `1024x1536`, `auto` |
| quality | `low`, `medium`, `high`, `auto` |

## Типичные промпт-паттерны

- **Фотореализм:** «photorealistic, 50mm lens, natural lighting, shallow depth of field»
- **Иллюстрация:** «flat illustration, minimalist, pastel palette, vector style»
- **Постер:** «poster design, bold typography, centered composition, high contrast»
- **Мокап/UI:** «clean product mockup on neutral background, soft shadow, studio lighting»

## Что НЕ делать

- **НЕ** запрашивать у пользователя API-ключ — он уже в `<your-knowledge-base>/.env`.
- **НЕ** светить ключ в логах/чате.
- **НЕ** генерировать под NDA/реальных людей без явного согласия — особенно для Yellow Rocks и закрытых клиентов.
- **НЕ** класть результат в публичные/git-отслеживаемые папки без явной просьбы (по умолчанию — `~/Downloads/`).

## Дебаг

- Ошибка `OPENAI_API_KEY не найден` → проверь `<your-knowledge-base>/.env` (`grep -l OPENAI` без `-h`, не светить значение).
- Ошибка `model not found` → модель могла смениться; переключи на `gpt-image-1` через `--model`.
- `billing_hard_limit_reached` → баланс/лимит на OpenAI.
