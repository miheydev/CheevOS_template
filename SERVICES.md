# Сервисы после перезагрузки

> Что нужно поднять после перезагрузки машины. Заполни под свой стек.

## Быстрый старт (всё разом)

```bash
cd ~/Documents/<your-workspace>

# Пример: локальные сервисы подпроектов
# cd <project-1> && ./scripts/start.sh && cd ..
# cd <project-2> && make run-dev && cd ..
```
